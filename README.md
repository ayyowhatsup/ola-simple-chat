## Ola

### Introduction

Simple chat application using Socket.io

### Setting up locally
```
git clone https://github.com/ayyowhatsup/ola-simple-chat

cd ola-simple-chat

npm install

npm run dev
```
> [!NOTE]
> Environment variables

```
DATABASE_URL=
CK_SECRET=
CK_HTTP_ONLY=
CK_SECURE=
CK_MAX_AGE=
```
